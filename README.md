# 前言

欢迎来到基于SSM的房屋租赁系统项目！本项目是一个基于Java语言，整合了Spring、Spring MVC和MyBatis框架的Web应用。通过这个项目，我们致力于实现便捷、高效的房屋租赁服务。以下是本项目的详细介绍。

## 内容介绍

基于SSM的房屋租赁系统主要包括以下几个模块：房屋信息发布、房屋查询、租赁订单管理、用户管理等。系统采用前后端分离的设计模式，前端使用Vue、JS和CSS3技术，后端则基于Java和SSM框架。通过本系统，用户可以轻松发布和查询房屋信息，实现线上租赁全流程。

## 技术介绍

- **语言：** Java
- **使用框架：** Spring、Spring MVC、MyBatis
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
